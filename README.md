# OCR-Vehicle-Data-Extractor
🚗 Android OCR application that extracts structured vehicle information from Malaysian Registration Certificates (Old &amp; New formats). Powered by 🔍 Google ML Kit OCR and 📷 CameraX, built with 💙 Kotlin, 🏗️ MVVM, ⚡ Coroutines &amp; 🎨 Material Design for fast, accurate document scanning and intelligent data extraction.
