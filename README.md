# OCR-Vehicle-Data-Extractor

# 🚗 Malaysian Car RC OCR

An Android application that automatically recognizes and extracts essential information from **Malaysian Vehicle Registration Certificates (RC)** using **Optical Character Recognition (OCR)**.

The application is designed to support both **Old** and **New** Malaysian RC formats, enabling users to quickly scan vehicle registration documents and automatically retrieve key information with minimal manual effort.

# ✨ Features

- 📷 Capture RC using the device camera
- 🖼️ Select RC image from the gallery
- 🔍 OCR-powered text recognition using Google ML Kit
- 📄 Supports both Old and New Malaysian RC formats
- ⚡ Fast and accurate extraction of vehicle information
- 🎯 Intelligent parsing and field mapping from OCR text
- 📱 Clean and intuitive Material Design interface
- ✅ Reduces manual data entry and human errors

---

# 📌 Extracted Information

The application can automatically identify and extract:

- Registration Number
- Owner Name
- Identification Number (NRIC/Company Registration)
- Chassis Number
- Engine Number
- Vehicle Make
- Vehicle Model
- Registration Date
- Engine Capacity
- Fuel Type
- Body Type
- Vehicle Colour
- Seating Capacity
- Vehicle Class
- Vehicle Status

> **Note:** The extracted information may vary depending on the RC format and OCR confidence.

# 🔄 Application Workflow

1. Capture an RC using the camera or choose one from the gallery.
2. Process the image using Google ML Kit OCR.
3. Detect and recognize all visible text.
4. Parse the recognized text to identify vehicle-specific fields.
5. Display the extracted information in a structured and user-friendly format.
